
BIKE SALES DASHBOARD

Overview

An interactive Excel dashboard built using a bike buyers dataset to explore customer demographics, purchasing behavior, and sales trends.

Tools Used
- Microsoft Excel

Excel Features Used
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Dashboard Design

Dashboard Insights
The dashboard analyzes:
- Average Income by Bike Purchase
- Customer Commute Distance
- Customer Age Groups
- Bike Purchase Trends

Interactive Filters
- Marital Status
- Region
- Education

Key Findings
- Higher-income customers are more likely to purchase bikes.
- Middle-aged customers represent the largest group of buyers.
- Commute distance influences purchasing decisions.
- Customer demographics impact sales trends.

Skills Demonstrated
- Data Cleaning
- Data Analysis
- Data Visualization
- Dashboard Development
